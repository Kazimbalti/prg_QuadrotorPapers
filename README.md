- [**Minimum Snap Trajectory**](http://www-personal.acfr.usyd.edu.au/spns/cdm/papers/Mellinger.pdf): Kumar's Minimum Snap trajectory generator. [Code](https://github.com/ethz-asl/mav_trajectory_generation)

- [**A computationally efficient motion primitive for quadrocopter trajectory generation**](https://flyingmachinearena.org/wp-content/publications/2015/mueTRO15.pdf): Raffaello Dâ€™Andrea's trajectory generator. [Code](https://github.com/ethz-asl/mav_trajectory_generation)
